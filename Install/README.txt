README

To get this project running quickly, stop any applications running on port 8080 or 8443, and follow these instructions:

WINDOWS

1. Double click threadfix.bat.

MAC OSX / Linux

1. Open Terminal / command shell
2. Navigate to this folder
3. Execute the command 'sh threadfix.sh'

Once the server has started normally, navigate to https://localhost:8443/threadfix/ in a web browser and log in with

username : user
password : password

From there you can add users as you please by going to configuration on the header bar and clicking manage users.
